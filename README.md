# Pizza-sales-Analysis

Topic 1 : Data Exploration

1. Retrieve the total numbers of orders placed.
2. Calculate the total revenue generated from pizzas.
3. Identify the highest priced pizzas.
4. identify the most common size pizza ordered.

Topic 2 : Sales Analysis

1. List the top 5 most ordered pizza types along with thier qunatities.
2. determine the disctribution of orders by hour of the day.
3. Determine top 3 most ordered pizza type based on revenue.

Topic 3 : Operational Insights

1. Calculate the percentage contribution of each pizza type to total revenue.
2. Analyze the cummulative revenue generated over time.
3. Determine the top 3 most ordered pizza types based on revenue for each pizza category.

Topic 4 : category Wise Analysis

1. Find the total quantity of each pizza category ordered.
2. Find the category wise distribution  of pizzas.
3. Group the orders by date and calculate the average number of pizzas ordered per day.
